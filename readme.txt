== Description ==

This plugin add a widget to show ChatMe.im user status

For more information, check out [chatme.im](http://chatme.im/) and [webchat](https://webchat.chatme.im/).

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload this directory to your plugins directory. It will create a 'wp-content/plugins/chatmeim-status-widget/' directory or install it via the WordPress.org plugin directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
